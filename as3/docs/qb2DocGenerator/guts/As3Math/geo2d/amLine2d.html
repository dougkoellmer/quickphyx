public function intersectsLine(otherLine:amLine2d, outputIntPoint:amPoint2d = null, distanceTolerance:Number = 0, radianTolerance:Number = 0):Boolean

Determines if two lines intersect. Overlapping lines return false.

Parameters

otherLine:amLine2d — The other line to check against.
 
outputIntPoint:amPoint2d (default = null) — Optional output for the intersection point. If non-null, intPnt's x and y will be modified by the function if the function returns true, otherwise it will be unmodified.
 
distanceTolerance:Number (default = 0) — The distance, for example, that the end point of one line segment can be from another line segment and still return true for intersection.
 
radianTolerance:Number (default = 0) — In the case of two infinite lines, this is the tolerance that determines whether they are parallel. If they are not parallel, they intersect.

Returns
Boolean — Whether or not there is intersection between the two lines.

isOverlappedBy()method 
public function isOverlappedBy(otherLine:amLine2d, radianTolerance:Number = 0, isOnTolerance:Number = 0, outputLine:amLine2d = null):Boolean

Determines if this and otherLine are colinear and overlap each other. This is determined differently depending on the lineType's involved. For example, if both lines are segments, then one line must have one of its endpoints on the other. However if both lines are infinite, then the only condition to be satisfied is colinearity.

Parameters

otherLine:amLine2d — The other line to test against.
 
radianTolerance:Number (default = 0) — The angular discrepancy that's allowed between the lines.
 
isOnTolerance:Number (default = 0) — The distance the two lines can be apart. Specifically, how far this line's point1 point can be from an imaginary infinite line passing through otherLine.
 
outputLine:amLine2d (default = null) — Optional output for the overlap found, if any. If non-null, outputLine is set with to the line representing the overlap. If no overlap is found, this line remains unchanged. Note that the outputLine's lineType can vary depending on the two lineType's tested. Also, if the lineType of outputLine is set to either LINE_TYPE_INFINITE or LINE_TYPE_RAY, then the line's distance from point1 to point2 is set to 1.

Returns
Boolean — Whether or not the two lines overlap.
