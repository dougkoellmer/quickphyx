angleproperty 
angle:Number  [read-only]

Returns the clockwise rotation in radians of this vector with respect to the egativey-axis, i.e. (0, -1). This axis is chosen as opposed to the more common (0, 1) because it corresponds to the default "up vector" of most display objects.
